# Wave Particles

A small Three.js learning project focused on creating a particle system using `BufferGeometry` and animating it as a wave over time.

The project explores how particle positions and colors are stored in buffer attributes and how updating those attributes each frame can be used to create wave-like motion.

<br/>

## â­ What this project does

- Creates a particle system using `THREE.BufferGeometry`
- Manually fills position and color buffers (`Float32Array`)
- Renders particles using `THREE.Points`
- Animates the particles vertically using a sine wave based on time and position
- Uses additive blending and textures to enhance visual appearance
